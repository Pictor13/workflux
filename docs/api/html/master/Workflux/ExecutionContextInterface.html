Workflux\ExecutionContextInterface

interface ExecutionContextInterface

ExecutionContextInterface reflects the current state of execution of a subject inside a state machine.

The StateMachineInterface relies on this interface to determine where to continue with a suspended state machine execution. It is also allows to share state machine variables across states in order to control flow.
