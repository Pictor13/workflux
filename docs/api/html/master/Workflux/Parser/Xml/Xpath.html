public DOMNodeList query($expression, DOMNode $context = null, $register_ns = null)

Takes an xpath expression and preprends the parser's namespace prefix to each xpath segment.

Then it runs the namespaced expression and returns the result. Example: '//statemachines/statemachine' - expands to -> '//wf:statemachines/wf:statemachine'
