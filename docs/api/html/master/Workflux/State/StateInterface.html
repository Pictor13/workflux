Workflux\State\StateInterface

interface StateInterface

StateInterface implementations are expected to act as a nodes that are part of a state graph.

The most important methods that allow to expose specific behaviour are "onEntry" and "onExit". Typically a state will manipulate the execution context of a given stateful subject in order to express intent.
