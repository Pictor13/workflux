workflux

Finite state machine for php.

View project on GitHub

Purpose

Workflux is a Finite State Machine(FSM) for php. It's supposed to be useful for implementing all kinds of FSM suiting scenarios like e.g. workflows or matching. Here are two reads on the topic, dealing with why/where you might want to use a FSM.

  • Why developers should be force-fed state machines
  • Why Developers Never Use State Machines

Requirements and installation

  • PHP v5.5+

Install the library via Composer:

./composer.phar require shrink0r/workflux [optional version]

Adding it manually as a vendor library requirement to the composer.json file of your project works as well:

{
    "require": {
        "shrink0r/workflux": "~0.5"
    }
}

Alternatively, you can download a release archive from the github releases.
