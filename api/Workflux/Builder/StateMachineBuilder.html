Workflux\Builder\StateMachineBuilder

class StateMachineBuilder implements StateMachineBuilderInterface

The StateMachineBuilder provides a fluent api for defining state machines.

The builder verifies the setup before creating the state machine, which makes it easier to spot errors when building automata.
