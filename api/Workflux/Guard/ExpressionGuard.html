Workflux\Guard\ExpressionGuard

class ExpressionGuard extends ConfigurableGuard

The ExpressionGuard employs it's verfification based on the evaluation of a given (symfony) expression.

The following variables are available within the configured expression: "subject" - The StatefulSubjectInterface that is being accepted/rejected. "params" - The parameters array of the subject's execution context.
